*Sistema de Busca Otimizado para Catálogo de Produtos - MegaStore*


**Descrição**
Este sistema utiliza a linguagem Rust e estruturas de dados de Tabelas Hash para resolver a lentidão nas buscas da MegaStore.

**Tecnologias**
* Rust 
* Standard Library (HashMap)

**Como Executar**

* Compilar: cargo build 

* Executar: cargo run 

* Testar: cargo test
